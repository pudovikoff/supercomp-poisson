# Решение задачи Пуассона методом фиктивных областей

[![C++](https://img.shields.io/badge/C++-11-blue.svg)](https://en.cppreference.com/w/cpp/11)

> Последовательная программа для решения двумерной задачи Дирихле для уравнения Пуассона в криволинейной области (вариант 7 - сапожок)

##  Описание

Реализация численного решения уравнения:
$$ - \Delta u = 1,  (x,y) \in  D \\
u = 0,    (x,y) \in \gamma $$

в области-сапожок: $$D = \{(x,y): -1 < x,y < 1} /\ \{(x,y): 0 \leq x,y < 1\} $$

### Шаги решения задачи:

- **Метод фиктивных областей** - сведение к задаче в прямоугольнике
- **Разностная схема** - пятиточечный шаблон на равномерной сетке
- **Метод сопряженных градиентов** - итерационное решение СЛАУ с диагональным предобуславливанием
