# Быстрый старт

## Задача: Вариант 7 - Область-сапожок

Решение уравнения Пуассона `-Δu = 1` в области:
```
D = {(x,y): -1 < x,y < 1} \ {(x,y): 0 < x,y < 1}
```
(квадрат минус правый верхний квадрант)

## За 3 команды

```bash
make                    # компиляция
./poisson_sequential    # запуск
python3 plot_solution.py  # визуализация (опционально, требует matplotlib)
```

## Ключевые особенности

✅ **Модульность**: геометрия отделена от численных методов  
✅ **Расширяемость**: легко изменить область (только `is_in_domain`)  
✅ **Эффективность**: O3 оптимизация, предобуславливание  
✅ **Надежность**: контроль монотонности, автоперезапуск  
✅ **Готовность**: база для OpenMP/MPI версий
